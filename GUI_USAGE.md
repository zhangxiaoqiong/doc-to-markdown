# 文档转换工具 - GUI使用指南

## 功能概览

- **文件选择**：支持DOCX、PDF、XLSX三种格式
- **拖拽上传**：直接拖拽文件到上传区域
- **实时进度**：显示转换进度和状态
- **质量控制**：设置质量阈值，自动识别待审核文件
- **结果导出**：导出转换结果和日志

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动应用

```bash
python gui_app.py
```

### 3. 选择文件

- **方式1**：点击"📁 选择文件"按钮
- **方式2**：直接拖拽文件到灰色区域

### 4. 配置设置

在"⚙️ 设置"选项卡中：
- 设置任务名称
- 选择输出目录
- 调整质量阈值（0-100）
- 启用自动重试和断点续传

### 5. 开始转换

点击"▶ 开始转换"按钮，转换将在后台运行。

### 6. 查看结果

- **进度**：实时显示转换进度
- **日志**：详细的操作日志
- **结果**：成功/失败/待审核统计

## 设置说明

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 任务名称 | 本次转换任务的标识 | 新建任务 |
| 输出目录 | 转换结果保存位置 | 用户主目录/转换结果 |
| 质量阈值 | 低于此评分的文件标记为待审核 | 75分 |
| 自动重试 | 失败文件自动重试 | 启用 |
| 断点续传 | 中断后可恢复转换 | 启用 |

## 导出功能

### 导出日志

点击"导出日志"按钮，保存所有操作日志到txt文件。

### 导出结果

点击"导出结果"按钮，生成HTML格式的结果汇总。

### 打开文件夹

点击"打开文件夹"直接在文件管理器中打开输出目录。

## 常见问题

### Q: 转换中途退出了怎么办？

A: 启用"断点续传"选项后，重新运行转换会自动跳过已完成的文件，继续处理失败文件。

### Q: 如何修改转换质量？

A: 在设置中调整"质量阈值"滑块，低于该评分的文件会被标记为待审核。

### Q: 支持哪些文件格式？

A: 支持DOCX（Word文档）、PDF（扫描件和原生PDF）、XLSX（Excel表格）。

### Q: 如何中断转换？

A: 点击"⏸ 暂停"按钮可以中断当前转换。如果启用了断点续传，下次运行可以继续。
